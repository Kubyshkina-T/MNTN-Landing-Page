Landing Page

This project is a responsive landing page created for learning purposes and front-end skill development.

The design is based on a Figma template and was used as a visual reference for building the layout and structure. During development, alternative fonts were selected in compliance with licensing requirements and suitable for web use, while preserving the overall visual style.

The main focus of this project was practical work with HTML, CSS, and JavaScript, as well as building an adaptive interface for different screen sizes.
